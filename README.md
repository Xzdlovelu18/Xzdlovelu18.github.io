# 熊志栋的个人博客

一个现代化、响应式的个人博客网站，展示个人信息、技能、工作经历和联系方式。

## 功能特性

- 🎨 **现代化设计** - 采用简洁、专业的设计风格
- 📱 **响应式布局** - 完美适配桌面、平板和移动设备
- ⚡ **平滑滚动** - 流畅的页面内导航体验
- 🎯 **动画效果** - 滚动触发的元素动画和技能条动画
- 🔧 **易于定制** - 模块化代码结构，方便内容和样式修改
- 📊 **数据可视化** - 技能水平和统计数据的直观展示
- 📞 **联系方式** - 多种联系方式和社交媒体链接

## 技术栈

- **HTML5** - 语义化标签和现代HTML特性
- **CSS3** - Flexbox、Grid布局、CSS变量和动画
- **JavaScript** - ES6+语法、DOM操作和事件处理
- **Font Awesome** - 图标库
- **响应式设计** - 媒体查询和流动布局

## 快速开始

### 安装

1. 克隆或下载项目文件到本地
2. 无需额外安装依赖，直接在浏览器中打开即可

### 使用

1. 使用文本编辑器打开项目文件
2. 修改 `index.html` 文件中的个人信息、技能、经历等内容
3. 根据需要调整 `style.css` 中的样式变量
4. 在浏览器中打开 `index.html` 文件查看效果

## 项目结构

```
├── index.html          # 主页面HTML文件
├── style.css           # 样式文件
├── script.js           # JavaScript交互逻辑
└── README.md           # 项目说明文档
```

## 自定义指南

### 修改个人信息

1. 打开 `index.html` 文件
2. 在对应的section中修改内容：
   - 导航栏：修改导航链接和品牌名称
   - 英雄区域：修改姓名、副标题和描述
   - 关于我：修改个人介绍文本
   - 技能：修改技能名称和水平
   - 经历：修改工作经历和教育背景
   - 联系方式：修改邮箱、电话和地址

### 调整样式

1. 打开 `style.css` 文件
2. 在 `:root` 中修改CSS变量：
   ```css
   :root {
       --primary-color: #4f46e5;  /* 主色调 */
       --text-primary: #1e293b;   /* 主要文本颜色 */
       --bg-primary: #ffffff;     /* 主要背景色 */
       /* 更多变量... */
   }
   ```

### 添加新功能

1. 参考现有的HTML结构添加新的section
2. 在 `style.css` 中添加对应的样式
3. 在 `script.js` 中添加交互逻辑（如果需要）
